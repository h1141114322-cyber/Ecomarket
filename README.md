# 🌿 EcoMarket

EcoMarket es una tienda web desarrollada con HTML5 y CSS3 que promueve el consumo responsable mediante la venta de productos ecológicos y sostenibles.

## 📋 Descripción

El proyecto fue desarrollado como evidencia académica aplicando HTML semántico, CSS3, Flexbox, Grid y diseño responsive, sin utilizar JavaScript.

## ✨ Funcionalidades

- Página de Inicio.
- Catálogo de productos.
- Página Sobre Nosotros.
- Página de Contacto.
- Diseño Responsive.
- Formularios con validación usando CSS.
- Efectos Hover y animaciones.
- Grid y Flexbox.
- Footer y navegación entre páginas.

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Git
- GitHub
